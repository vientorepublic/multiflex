# Multi Flex

Nest.js based file upload/download server implementation that supports multi-threading.

# Requirements

- Node.js 20.x
- MySQL or MariaDB Database Server

# Config

Create `.env` and add below:

```
PORT=
UPLOAD_PATH=
EXPIRES=86400000
RECAPTCHA_SECRET=
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_DATABASE=
```
